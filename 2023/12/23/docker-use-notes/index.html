常用操作

  1. 镜像操作：
  • docker images：查看本地的Docker镜像列表。
  • docker pull image_name：下载一个Docker镜像。
  • docker rmi image_name：删除一个本地Docker镜像。
  1. 容器操作：
  • docker ps：查看正在运行的容器列表。
  • docker ps -a：查看所有容器列表，包括已停止的容器。
  • docker run image_name：运行一个新的容器。
  • docker start container_name/id：启动一个已停止的容器。
  • docker stop container_name/id：停止一个运行中的容器。
  • docker restart container_name/id：重启一个容器。
  • docker rm container_name/id：删除一个已停止的容器。
  • docker exec -it container_name/id bash：在容器内部启动一个交互式Shell会话。
  • docker logs container_name/id：查看容器的日志。
  1. 网络操作：
  • docker network ls：查看Docker网络列表。
  • docker network create network_name：创建一个新的Docker网络。
  • docker network connect network_name container_name/id：将容器连接到指定的网络。
  • docker network disconnect network_name container_name/id：将容器从网络中断开连接。
  1. 数据卷操作：
  • docker volume ls：查看Docker数据卷列表。
  • docker volume create volume_name：创建一个新的Docker数据卷。
  • docker volume rm volume_name：删除一个Docker数据卷。
  • docker run -v volume_name:/container/path image_name：运行一个容器并挂载数据卷。
  1. 镜像构建和发布：
  • docker build -t image_name .：基于Dockerfile构建一个新的镜像。
  • docker push image_name：将一个本地镜像推送到Docker镜像仓库。
  1. 其他常用命令：
  • docker info：查看Docker系统信息。
  • docker version：查看Docker版本信息。
  • docker-compose：使用Docker Compose管理多个容器应用
